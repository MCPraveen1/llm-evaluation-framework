# LLM Evaluation Framework

## Overview
A modular framework for evaluating and comparing LLMs on the SST-2 sentiment analysis benchmark using Groq-hosted models.

## Features
- SST-2 dataset evaluation
- Multi-model comparison
- Accuracy, Precision, Recall, F1 metrics
- Reproducible sampling
- Modular architecture

## Models Tested
- Llama 3.1 8B Instant
- Llama 3.3 70B Versatile
